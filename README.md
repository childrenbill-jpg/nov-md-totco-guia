## Cómo subir esta actualización
1. Ir a https://github.com/childrenbill-jpg/nov-md-totco-guia
2. Click en **Add file → Upload files**
3. Arrastrar los 5 archivos del ZIP
4. **Commit changes**
5. Esperar 1-2 minutos y probar con Ctrl+F5
6. Los técnicos con la app instalada se actualizan solos (Service Worker v6)
